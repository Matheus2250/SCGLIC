Importar da Planilha Excel
Utilize esta funcionalidade para importar registros de uma planilha Excel. Certifique-se de que a planilha tenha os cabeçalhos corretos conforme a especificação abaixo.
Instruções:
  1. Prepare uma planilha Excel com os campos abaixo
  2. Selecione o arquivo usando o botão "Escolher arquivo"
  3. Clique em "Importar" para iniciar o processo
Cabeçalhos obrigatórios:
  • NUP
  • OBJETO
  • MODALIDADE
  • SITUAÇÃO
  • ÁREA DEMANDANTE
  • PREGOEIRO
  • TIPO
  • VALOR ESTIMADO
  • VALOR HOMOLOGADO
  • ECONOMIA
  • DT ABERTURA
  • DT HOMOLOGAÇÃO
Somente arquivos .xlsx ou .xls são aceitos.
Cuidado: Isso excluirá TODOS os registros existentes.
